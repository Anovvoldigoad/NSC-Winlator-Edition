================================================================================

WINLATOR (ANDROID):
  1. Create container
  2. Install .NET 8 runtime
  3. Copy published files to C:\NSC.Winlator\
  4. Run NSC.Winlator.exe

WINE/PROTON (LINUX):
  1. Install .NET 8 runtime
  2. Extract published files
  3. Run via wine NSC.Winlator.exe
  4. Configure wine path mapping
