  8. Download dari "Artifacts"

CARA 2: TERMUX (PALING POWER - TAPI AGAK RUMIT)
────────────────────────────────────────────────────────────────────────────
Waktu: 15 menit
Yang perlu: HP Android + Termux app
Langkah:
  1. Install Termux dari Google Play
  2. Buka Termux
  3. Paste ini:
     
     pkg update && pkg upgrade -y
     pkg install -y git wget unzip
     pkg install -y dotnet-runtime dotnet-sdk

  4. Tunggu ~10 menit (install dependencies)
  5. Download project:
     
     mkdir ~/projects
     cd ~/projects
     wget https://github.com/YOUR_USERNAME/NSC-Winlator-Edition/archive/main.zip
     unzip main.zip
     cd NSC-Winlator-Edition-main

  6. Build:
     
     cd src/NSC.Winlator
     dotnet restore
     dotnet build -c Release

  7. Tunggu ~3 menit build
  8. Output ada di: bin/Release/net8.0-windows/NSC.Winlator.exe
Hasil: File executable siap pakai!

✅ REKOMENDASI SAYA: CARA 1 (GITHUB)

Kenapa?

  ✓ Paling cepat
  ✓ GitHub Actions handle build
  ✓ Download siap pakai
  ✓ Tidak perlu install apapun
  ✓ Bisa dari HP saja!

Langkah detail:

STEP 1: Download File
────────────────────────────────────────────────────────────────────────────
1. Download: NSC-Winlator-Edition-Complete.zip
2. Extract di HP (gunakan file manager)
3. Lihat folder yang diekstrak


STEP 2: Buat GitHub Repo
────────────────────────────────────────────────────────────────────────────
1. Buka: https://github.com (login dulu)
2. Click "+" di kanan atas
3. Pilih "New repository"
4. Nama: NSC-Winlator-Edition
5. Deskripsi: Mod Manager untuk Naruto Storm Connections
6. Pilih "Public"
7. Click "Create repository"


STEP 3: Upload Semua File
────────────────────────────────────────────────────────────────────────────
1. Di repo GitHub yang baru dibuat
2. Click "uploading an existing file"
3. ATAU click "Add file" → "Upload files"
4. Buka file manager HP
5. Pilih semua file dari folder yang diekstrak
6. Drop ke GitHub upload area
7. Click "Commit changes"


STEP 4: GitHub Actions Build Otomatis!
────────────────────────────────────────────────────────────────────────────
1. Tunggu ~2 detik
2. Lihat tab "Actions"
3. Lihat workflow "Build NSC Winlator Edition" berjalan
4. Status akan berubah jadi ✓ dalam 2-3 menit
5. Selesai build!


STEP 5: Download Hasil Build
────────────────────────────────────────────────────────────────────────────
1. Buka tab "Actions" 
2. Click workflow terbaru
3. Scroll ke bawah → "Artifacts"
4. Download: NSC-Winlator-win-x64
5. File .zip berisi executable!


STEP 6: Extract & Gunakan
────────────────────────────────────────────────────────────────────────────
1. Extract artifact zip
2. Isi: NSC.Winlator.exe (+ semua file dependencies)
3. Copy ke PC atau Winlator
4. Run!
